# **App Name**: AdCraft AI

## Core Features:

- Image Upload: Allows users to upload vehicle images from their device.
- AI Ad Generation: Analyzes vehicle images and user intent to generate 'for sale' or 'wanted' ad drafts using a tool to tailor the output for the selected category and context.
- Ad Editing: Provides an interface for users to edit and refine the AI-generated ad text.
- Ad Saving: Enables users to save generated and edited ads within the app.
- Monetization: Offers premium ad features and subscription tiers.

## Style Guidelines:

- Primary color: Rich, saturated blue (#2962FF) evoking trust and reliability, appropriate for an application dealing with sales.
- Background color: Light, desaturated blue (#E5EAFE) to provide a clean and unobtrusive backdrop.
- Accent color: Vibrant violet (#7A00D9) for highlights and calls to action, providing contrast.
- Headline Font: 'Space Grotesk' sans-serif, for headlines. Body font: 'Inter' sans-serif, for body text.
- Simple, geometric icons representing vehicle features and ad options.
- Clean, card-based layout for displaying vehicle images and ad drafts.
- Subtle transitions and animations for a smooth user experience during ad generation and saving.
